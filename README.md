🎮 Phoneme Puzzle — Language Technology Game
📌 Overview

Phoneme Puzzle is an interactive educational web game designed to help users understand phonemes and their role in speech and pronunciation.

The player is given a word and must arrange the correct IPA (International Phonetic Alphabet) symbols in the correct order to match the word’s pronunciation.

🎯 Objective
Learn how words are broken into phonemes
Practice phonetic transcription using IPA symbols
Improve understanding of speech and language technology concepts
🧠 Concept (AI / Language Technology)

This game is related to:

Speech Processing
Phoneme Recognition
Natural Language Processing (NLP)
Linguistics & Phonetics

It simulates how systems process and analyze spoken language at the phoneme level.

🎮 How to Play
A word appears on the screen (e.g., CAT)
You are given shuffled phoneme pieces (e.g., /k/, /æ/, /t/)
Click a phoneme, then click a slot to place it
Arrange all phonemes in the correct order
Click "Check Answer"
💡 Game Features
✅ Multiple levels (8 levels with increasing difficulty)
🎯 Score system
💡 Hint system (limited hints available)
🔄 Reset and retry options
📊 Progress bar
🎨 Clean and simple UI
🧩 Example

Word: CAT
Correct Answer:

/k/ /æ/ /t/
